## Usage 

### server 
- cd server 
- python server
### client  
- Install the command yarn using : 
```
npm install -g yarn
```
- cd client
- Run ``` yarn install ``` to install project dependencies
- Start the project with the command ``` yarn start ```.
- Open in your browser: http://127.0.0.1:4000/
